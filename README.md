# h5-vue-vant

## Usage

This is a project template for [vue-cli](https://github.com/vuejs/vue-cli). **It is recommended to use npm 3+ for a more efficient dependency tree.**

``` bash
$ npm install -g vue-cli
$ vue init qqzylx/h5-vue-vant my-project
$ cd my-project
$ npm install
$ npm run dev
```
